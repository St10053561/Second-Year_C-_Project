## Project Overview for Console App

The project is a recipe app developed using Visual Studio and C# programming language. It has five files that contain different classes responsible for performing specific tasks.

- **Project:** This class serves as the entry point of the program and is responsible for running it.
- **Recipe:** The core of the program, this class stores and manipulates recipe data.
- **Task:** This class gets and captures information from users.
- **RecipeDescription:** This class stores recipe name and steps values to display the steps based on recipes.
- **CheckingCalories:** A test that checks if my project can show an error message if the calories exceed 300.

## How to Run the Program

To run the program, follow these steps:

1. Unzip "ST10053561_Venkata Jannatha_PROG6221_Part2" folder by right-clicking on it and selecting "Unzip".
2. After unzipping, open "POE_Project.sln" file in Visual Studio software.
3. Once you've opened your solution in Visual Studio, navigate to `Program.cs` file where you will see "//I link here in this program to run Recipe show = new Recipe();" line. 
4. Simply click on 'Start' button with green arrow at top of IDE window to launch application.
